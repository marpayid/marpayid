# **App Name**: MarPay

## Core Features:

- Global Search Interface: An interactive top search bar optimized for mobile navigation and rapid product discovery.
- Responsive Banner Slider: Touch-swipeable hero section for promotional offers and latest marketplace updates.
- Category Quick-Menu: A dense icon-based grid for fast navigation across different product sectors.
- Flash Sale Module: A high-conversion section featuring timed discount counters and limited stock progress bars.
- Optimized 2-Column Product Grid: A high-density mobile layout showcasing product images, prices, and ratings side-by-side.
- Seamless Cart and Checkout: A intuitive multi-step purchasing flow designed for mobile checkout speed.
- Persistent Bottom Navigation: Quick-access menu for Home, Feed, Orders, and Profile, sticking to the base of the screen.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5) evoking reliability and a modern digital finance aesthetic.
- Background color: A very soft, tinted grey (#F7F8FD) that shares the indigo hue while maintaining a clean light mode.
- Accent color: Vibrant Sky Blue (#0EA5E9), used sparingly for conversion actions and visual emphasis.
- Font pairing: 'Poppins' for expressive headlines and branding, and 'Inter' for high readability in product details and pricing lists.
- Consistent thin-line vector icons to maintain a sophisticated and clean mobile-first interface.
- Edge-to-edge mobile-first design with 16px standard padding and 8px card gutters for a balanced density.
- Subtle slide-up transitions for modal screens and smooth x-axis scrolling for promotional banners.
